01. Arquitectura Cromática y Fundamentos Visuales

1.1 Paleta Primaria: "Azul Petróleo"

Seleccionada como el identificador central de Recetar para establecer una diferenciación competitiva frente a soluciones existentes (evitando azul genérico o tonos púrpuras) y transmitir credibilidad médica sin caer en estéticas frías u hospitalarias.

1.2 Paleta Secundaria: Teal. Acento

Criterio de Aplicación: No tiene responsabilidades informáticas o de estado. Su uso queda exclusivamente para componentes de acción e hitos interactivos (FABs, toggles, stepper activo) para evitar competir con la jerarquía del azul petróleo.

3.1 Regla de Aplicación (Ley de Prägnanz)

Regla de Aplicación (Ley de Prägnanz): El color de rol actúa como un acento, no como un tema global. La estructura visual base (grillas, layouts, cabeceras, espaciados) es idéntica para los 3 roles. Garantiza que el sistema se perciba como un todo coherente.

3.3 Matriz de Comportamiento en Componentes UI

✅ Sí adoptan color de rol Fondo de ítems activos en la barra de navegación lateral (Sidebar), anillo perimetral del avatar de usuario, y etiqueta contextual en la cabecera.
❌ No adoptan color de rol Botones primarios (acción con lenguaje del sistema, no del rol), tablas, listas de datos y alertas críticas de riesgo clínico.

04. SISTEMA TIPOGRÁFICO

La selección tipográfica responde a la necesidad de garantizar una lectura óptima en pantallas con cierta complejidad de datos técnicos (principio activo, matrículas profesionales y códigos comerciales).

4.1 Fuentes Oficiales

Inter (Tipografía Principal): Elegida por su rendimiento en interfaces, presenta una altura de x elevada para facilitar la lectura de textos con mayúsculas y minúsculas. Se diferencia del estándar por defecto (Roboto) del resto de las interfaces.
JetBrains Mono (Tipografía de Datos): Destinada exclusivamente a cadenas de caracteres (códigos de recetas, identificadores de CUIL, números de matrículas profesionales y nomencladores).

05. ESPACIADO, GEOMETRÍA Y ESTRUCTURA DE LAYOUT

5.1 Escala de Espacio (Base 4px)

Alineación pixel-perfect compatible con la grilla de 8px Angular Material. Todos los tokens son múltiplos de 4px.
